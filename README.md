⚙️ Installation & Setup
# Clone repo
git clone https://github.com/your-username/admin-dashboard.git

# Move into project
cd admin-dashboard

# Install dependencies
npm install

# Run development server
npm run dev
🏗️ Build for Production
npm run build

📌 Notes
Make sure all screenshots are placed inside /screenshots folder.
